# Introduction to Python for Bioinformatics
A beginner bioinformatics project using Phython to learn core programming concepts, CSV data analysis, and basic plotting.
This repository is a beginner-level bioinformatics learning project designed to build foundational Python skills for biological data analysis.
The project focuses on understanding basic Python programming, working with biological data stored in CSV files, and visualizing data using simple plots.
# Project Objectives
- Learn core Python programming concepts
- Understand how to read and manipulate CSV data using pandas
- Create basic data visualizations using matplotlib
- Build confidence using Jupyter notebooks and GitHub
# Repository Structure
- **01 Basic python.ipynb**  
  Covers Python fundamentals such as variables, lists, loops, and functions.
- **02 Reading csv with pandas.ipynb**  
  Demonstrates how to read CSV files, explore datasets, and perform basic filtering using pandas.
- **03 Basic plots.ipynb**  
  Introduces basic data visualization techniques including line plots and bar charts using matplotlib.
# Tools and Technologies used
- Python
- Google Colab / Jupyter Notebook
- pandas
- matplotlib
- GitHub
# Target Audience
- Beginners learning Python
- Students interested in bioinformatics
- Life science students transitioning into computational biology
